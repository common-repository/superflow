== How it works ==

**No-code installation:** Install the Superflow wordpress plugin without having to manually embed a snippet code.

This section describes how to install the plugin and get it working:

1. **Install plugin:** Install the Superflow plugin from the Wordpress marketplace.
2. **Access Plugin Settings:** After installation is complete, go to Superflow plugin settings.
3. **Connect with Superflow:** Click the ‚ÄòConnect Superflow‚Äô button.
4. **Superflow account setup and linking:** New Users: If you don‚Äôt have a Superflow account, a guided setup process will commence. Follow the step-by-step instructions to create a new & link your account. Existing Users: If you already have a Superflow account, the system will automatically link it to your WordPress account.
5. **Verify Installation:** Now the Superflow script is installed on your website. It will then open your wordpress website with the Superflow toolbar installed.
6. **Add your first comment:** Now you can start leaving comments on top of your live website!
